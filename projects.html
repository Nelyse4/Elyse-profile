1. MyHealthData Application

This project helps to track and categorize patients' data, recording essential information. This tool assists in keeping patient data, determining BMI, maximum heart rate, and calculating patient age in days. It provides a basic health description of the patient.

2. Valid Baggage Application

An app that allows valid baggage at the airport to proceed for being carried. If it exceeds the required weight and size, there is an additional fee. Once paid, the bag proceeds. The app helps the user to know the weight and size of valid baggage.

3. Book catalogue system
